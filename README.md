# Hospital de Teresina — Sistema CRUD em Python

Trabalho Final da disciplina de python.
Sistema CRUD de pacientes executado via terminal .

## Estrutura dos arquivos

| Arquivo    | Responsabilidade                                            |
|------------|-------------------------------------------------------------|
| `main.py`  | Menu principal e loop de execução                           |
| `crud.py`  | Funções de criar, listar, buscar, editar, excluir e relatório |
| `utils.py` | Validações, leitura segura de entradas e função geradora    |
| `dados.py` | Tupla de especialidades, lista de pacientes e set de CPFs   |
